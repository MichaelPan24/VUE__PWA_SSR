# vue-pwa-ssr

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### ssr 构建
```
npm run ssr:build
```

### ssr开发服务器
```

npm run ssr:serve
```

### ssr 生产模式启动
```

npm run ssr:start
```
